# Digimon Scraper Setup Guide

All required packages are already in your `requirements.txt`:

## 🎯 What Each Script Does

### 1. scrape_links.py
**Purpose:** Extract Digimon profile URLs from the Game8 list page

**Input:** `/data/raw/game8/554944.html` (already downloaded)

**Output:** `/data/processed/digimon-links.json`
```json
{
  "Kuramon": "https://game8.co/games/Digimon-Story-Time-Stranger/archives/555193",
  "Pagumon": "https://game8.co/games/Digimon-Story-Time-Stranger/archives/555166",
  ...
}
```

**Run:** `python scraper/game8/scrape_links.py`

---

### 2. scrape_requirements.py
**Purpose:** Scrape evolution requirements from each Digimon's Game8 profile

**Input:** `/data/processed/digimon-links.json` (generated by step 1)

**Output:** `/data/processed/digimon-requirements.json`

**What it extracts:**
- General requirements (Agent Rank, Level, etc.)
- DNA Digivolution (multiple Digimon + personality conditions)
- Mode Changes (evolution from another form)

**Example output:**
```json
{
  "Omnimon": {
    "agent_rank": 8,
    "paths": [
      {
        "type": "dna",
        "conditions": [
          {"digimon": "WarGreymon", "requirement": "Daring"},
          {"digimon": "MetalGarurumon", "requirement": "Friendly"}
        ]
      }
    ]
  },
  "Kuramon": null
}
```

**Features:**
- 1 second delay between requests (respects Game8)
- Automatic retry with backoff on failures
- Detailed logging of progress
- Can be resumed with `--start N` and `--limit M` flags

**Runtime:** ~7-10 minutes for ~450 Digimon

**Run:** `python scraper/game8/scrape_requirements.py`

---

### 3. merge.py
**Purpose:** Integrate extracted requirements into your main `digimon.json`

**Input:** 
- `/data/processed/digimon.json` (original)
- `/data/processed/digimon-requirements.json` (extracted)

**Output:** `/data/processed/digimon.json` (updated)

**Features:**
- Creates automatic backup: `digimon.json.backup`
- Fuzzy name matching (handles spacing/capitalization)
- Detailed statistics on matches
- Sample verification of merged data

**Run:** `python scraper/game8/merge.py`

---

### 4. run_pipeline.py
**Purpose:** Run all three steps in sequence (RECOMMENDED)

**Advantages:**
- Single command to run entire pipeline
- Automatic error handling between steps
- Clear progress reporting
- Easy to test with `--limit` flag

**Usage:**
```bash
# Complete pipeline
python scraper/game8/run_pipeline.py

# Test with 10 Digimon
python scraper/game8/run_pipeline.py --limit 10

# Resume from Digimon #100, process 50 more
python scraper/game8/run_pipeline.py --start 100 --limit 50
```

## 🔄 Resume from Checkpoint

If scraping is interrupted, you can resume:

```bash

python scraper\game8\run_pipeline.py --start 100 --limit 100
```

The `digimon-requirements.json` file will be appended/overwritten.

## 🐛 Common Issues

| Issue | Solution |
|-------|----------|
| `ModuleNotFoundError: No module named 'requests'` | Run `pip install -r requirements.txt` |
| `Failed to fetch URL` (repeated) | Game8 is rate limiting - increase `REQUEST_DELAY` |
| `No requirements section found` | Normal for base forms - they'll have `requirements: null` |
| `Unmatched Digimon` in merge | Name mismatch - check the warning logs |
| `HTML file not found` | Make sure `554944.html` is in `/data/raw/game8/` |

## 📚 Understanding the Output

### requirements object structure:

```
{
  "agent_rank": 8,           ← Optional: minimum Agent Rank
  "level": 30,               ← Optional: minimum level
  "paths": [                 ← Array of evolution methods
    {
      "type": "dna",
      "conditions": [
        { "digimon": "Name", "requirement": "Condition" }
      ]
    },
    {
      "type": "mode_change",
      "from": "BaseName"
    }
  ]
}
```

If there are no evolutions: `"requirements": null`
